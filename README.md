### 3D Portfolio Website Project

This repository hosts the source code for my interactive 3D developer portfolio website, designed to showcase my credentials, project gallery, and professional experience.

* **Live Deployment:** [kinshukkhandelwal.qzz.io](https://kinshukkhandelwal.qzz.io)
* **Core Stack:** React 19, Three.js, React Three Fiber, GSAP, Framer Motion, Lenis, Tailwind CSS v4, EmailJS.
* **Key Features:**
  * Interactive 3D particle staging and stage meshes.
  * Fully responsive design with Dark/Light theme toggle controls.
  * Direct contact form integration utilizing EmailJS API (with custom SPF/DMARC routing).

#### Local Development Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KINSHUKHERE/Personal-Portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add the following keys:
   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Start the local server:**
   ```bash
   npm run dev
   ```
