## 🌌 My 3D Portfolio Website Project

This repository hosts my fully interactive 3D developer portfolio website.

🔗 **Live Link:** [kinshukkhandelwal.qzz.io](https://kinshukkhandelwal.qzz.io)

### ✨ Key Features:
* **Interactive 3D Backgrounds:** Dynamic mesh & particle stage elements built with **Three.js**, **React Three Fiber**, and **Drei**.
- **🌗 Dark / Light Mode Support:** Dual themes utilizing custom HSL color palettes.
- **📜 Lenis Smooth Scroll:** Natural, fluid web scrolling page transitions.
- **📈 GSAP & Framer Motion:** Sophisticated micro-animations and reveal-on-scroll states.
- **✉️ EmailJS Contact Form:** Fully functional secure email system with direct redirection and custom template.

### 🚀 Local Setup Instructions:
1. **Clone the Repo:** `git clone https://github.com/KINSHUKHERE/Personal-Portfolio.git`
2. **Install Dependencies:** `npm install`
3. **Environment Setup:** Add your EmailJS keys inside `.env`:
   ```env
   VITE_EMAILJS_SERVICE_ID=your_id
   VITE_EMAILJS_TEMPLATE_ID=your_id
   VITE_EMAILJS_PUBLIC_KEY=your_key
   ```
4. **Run Dev Server:** `npm run dev`
